# Página de visualización y filtrado de candidatos


## Objetivos

- Visualización de una lista de candidatos
- Filtrado por varios atributos
- Datos precargados de un archivo .json


## Instalación y ejecución con docker

- Construir y levantar la aplicación

  `docker-compose up -d --build`

- Acceder a la aplicación en tu navegador
  
  [http://localhost:3000](http://localhost:3000) 


## Ejecutar pruebas unitarias

  `docker exec -it visualizacion-filtrado-app npm test`
